# 3D Лабиринт на OpenGL

Интерактивный 3D лабиринт от первого лица, написанный на C++ с использованием OpenGL и GLUT.

![OpenGL](https://img.shields.io/badge/OpenGL-GLUT-blue)
![C++](https://img.shields.io/badge/C++-17-green)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

## Возможности

-  Вид от первого лица с плавным движением
-  Мерцающий факел — динамическое освещение
-  Туман — атмосфера тёмного подземелья
-  Миникарта в углу экрана
-  Прицел по центру
-  Три лабиринта разной сложности
-  Детекция финиша с экраном победы
-  Стрейф влево/вправо

## Управление

| Клавиша | Действие |
|---------|----------|
| W / S | Вперёд / Назад |
| A / D | Поворот влево / вправо |
| Q / E | Стрейф влево / вправо |
| ESC | Выход |

## Меню

| Клавиша | Действие |
|---------|----------|
| W / S | Выбор лабиринта |
| 1 / 2 / 3 | Быстрый выбор |
| ENTER | Старт |

## Установка и запуск

### Требования

- Visual Studio 2019 / 2022
- NuGet пакет: `nupengl.core`

### Сборка

1. Клонируй репозиторий:
```bash
git clone https://github.com/KoDiK2005/Labirint.git
```

2. Открой `Labirint.sln` в Visual Studio

3. Установи зависимости через NuGet:
   - ПКМ по проекту → **Manage NuGet Packages**
   - Найди и установи `nupengl.core`

4. Собери и запусти (`Ctrl+F5`)

## Технологии

- **C++** — основной язык
- **OpenGL** — рендеринг 3D графики
- **GLUT (freeglut)** — окно, ввод, таймер
- **Модель освещения Phong** — диффузный свет + specular
- **GL_FOG** — экспоненциальный туман
